GPT-5가 정식 출시됐습니다. 주요 변경점은 다음과 같습니다.

## 핵심 변경사항

- **멀티모달 강화**: 이미지·오디오 입력 품질 향상
- **추론 능력**: 수학·코딩 벤치마크에서 기존 대비 30% 이상 개선
- **컨텍스트 윈도우**: 256k 토큰으로 확장

## 백엔드 개발자 관점

API 스펙은 기존 GPT-4와 호환되므로 모델명(`gpt-5`)만 변경하면 됩니다. 단, 응답 품질이 높아진 만큼 프롬프트 구조를 재검토할 가치가 있습니다. 특히 Function Calling 정확도가 크게 개선된 점이 눈에 띕니다.
